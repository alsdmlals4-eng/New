# Contributing

`my little boat`는 작은 보트에서 바다를 감상하는 Godot 기반 1인칭 힐링 표류 게임입니다. 모든 기여는 이 핵심 경험을 해치지 않는 방향이어야 합니다.

## 작업 전 확인

1. 관련 Issue가 있는지 확인합니다.
2. 없으면 먼저 Issue를 만듭니다.
3. 변경 범위를 작게 잡습니다.
4. Godot에서 열 수 있는 프로젝트 구조를 유지합니다.
5. 모바일 세로 화면을 먼저 고려합니다.

## 코드 원칙

- Godot 4.7 stable과 GDScript를 사용합니다.
- 초보자가 읽을 수 있는 명확한 이름을 사용합니다.
- 씬과 스크립트는 가능한 한 한 가지 책임만 갖게 합니다.
- 노드 이름은 역할을 알 수 있게 짓습니다.
- 중요한 함수에는 짧은 주석을 둡니다.
- 온라인, 결제, 광고, 유료 에셋 의존을 추가하지 않습니다.

## 게임 방향

추가해도 좋은 것:

- 마음 선택에 따른 색감, 소리, 이벤트 변화
- 사진, 편지, 풍경, 동반자 반응 수집
- 감상모드와 방치형 감상 경험 개선
- 모바일 세로 화면 UI 개선
- 조용한 자연음과 플레이어의 노래 목록

추가하지 말아야 할 것:

- 전투
- 실패 조건
- 경쟁 점수
- 결제
- 광고
- 온라인 편지 공유
- 복잡한 상점

## 테스트

Pull Request 또는 커밋 전 아래를 확인합니다.

- Godot에서 `project.godot`을 열 수 있는가
- 실행 시 `scenes/main_menu.tscn`이 시작되는가
- 마음 선택 후 `scenes/game.tscn`으로 이동하는가
- 사진찍기, 감상모드, 속도조절 버튼이 작동하는가
- 편지 또는 풍경 기록이 앨범에 반영되는가
- 모바일 세로 화면에서 주요 버튼이 잘 보이는가
- PC 마우스 입력을 고려했는가
