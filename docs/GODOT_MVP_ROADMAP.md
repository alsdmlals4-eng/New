# Godot MVP Roadmap

`my little boat` Godot MVP를 작은 단계로 나누어 진행합니다.

## 1단계: 프로젝트 골격

완료 기준:
- `project.godot`이 존재한다.
- `scenes/main_menu.tscn`, `scenes/game.tscn`, `scenes/album.tscn`이 존재한다.
- `scripts/core/`, `scripts/ui/`, `scripts/voyage/` 구조가 있다.
- README와 협업 문서가 Godot 기준으로 정리되어 있다.

## 2단계: 마음 선택

완료 기준:
- 평온, 지침, 외로움, 설렘 버튼이 보인다.
- 선택한 마음이 `GameState`에 저장된다.
- 선택 후 게임 화면으로 이동한다.

## 3단계: 항해 화면

완료 기준:
- 플레이어 몸은 보이지 않는다.
- 보트 앞부분, 바다, 하늘을 감상하는 구도가 있다.
- 모바일 세로 화면에서 버튼이 잘 보인다.
- PC 마우스 입력과 모바일 터치 입력을 고려한다.

## 4단계: 핵심 조작

완료 기준:
- 사진찍기 버튼이 기록을 남긴다.
- 감상모드 버튼이 UI 상태를 바꾼다.
- 속도조절이 느림, 보통, 빠름으로 순환한다.

## 5단계: 5분 항해 루프

완료 기준:
- 5분 타이머가 동작한다.
- 타이머 종료 후 오늘의 항해 기록 문구가 표시된다.
- 종료 후에도 방치형 감상 모드로 계속 머무를 수 있다.

## 6단계: 편지, 풍경, 동반자

완료 기준:
- 병 속 편지 문구가 로컬에서 랜덤으로 나온다.
- 일출, 비, 고래, 밤바다 같은 풍경 기록이 가능하다.
- 동반자 호감도 Lv 1~3이 기록 수에 따라 변한다.

## 7단계: 앨범

완료 기준:
- 사진 앨범, 풍경 앨범, 편지 보관함 구조가 있다.
- 각 기록 수와 최근 기록이 보인다.
