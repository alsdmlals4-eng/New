# Godot MVP Roadmap

앞으로 만들 후보 장르는 비주얼노벨, 뱀서라이크, 타워디펜스입니다. 모두 소규모 MVP로 시작합니다.

## 1. 비주얼노벨 MVP

목표:
- 짧은 이야기 1개를 끝까지 플레이할 수 있게 만듭니다.

핵심 기능:
- 타이틀 화면
- 대사창
- 캐릭터 이름 표시
- 배경 이미지 교체
- 선택지 2개 이상
- 엔딩 2개 이상
- 대사 로그
- 로컬 저장 / 불러오기

MVP에서 제외:
- 보이스
- 복잡한 호감도 시스템
- 온라인 공유
- 유료 에셋 의존

## 2. 뱀서라이크 MVP

목표:
- 5분 동안 버티며 성장 선택을 경험하는 작은 전투 루프를 만듭니다.

핵심 기능:
- 플레이어 이동
- 자동 공격 1종
- 적 스폰
- 경험치 수집
- 레벨업 선택지 3개
- 간단한 보스 또는 마지막 웨이브
- 결과 화면

MVP에서 제외:
- 복잡한 스킬 트리
- 많은 캐릭터
- 온라인 랭킹
- 결제 / 광고

## 3. 타워디펜스 MVP

목표:
- 한 맵에서 타워를 배치하고 웨이브를 막는 기본 구조를 만듭니다.

핵심 기능:
- 고정 경로
- 타워 배치
- 타워 3종
- 적 3종
- 웨이브 5개
- 자원 획득과 사용
- 승리 / 패배 화면

MVP에서 제외:
- 맵 에디터
- 복잡한 업그레이드 트리
- PvP
- 결제 / 광고

## 추천 순서

1. 비주얼노벨
2. 타워디펜스
3. 뱀서라이크

이 순서를 추천하는 이유는 UI와 저장 구조를 먼저 익히고, 그 다음 배치와 웨이브, 마지막으로 실시간 액션과 최적화를 다루기 좋기 때문입니다.
