# Codex Goals

Codex에게 작업을 맡길 때 아래 형식을 사용하면 좋습니다.

## 예시 1: 마음 선택 개선

```text
Godot 프로젝트에서 main_menu.tscn의 마음 선택 UI를 개선해줘.
모바일 세로 화면 기준으로 버튼 간격이 안정적이어야 하고,
선택 후 game.tscn으로 이동해야 해.
README 테스트 체크리스트도 갱신해줘.
```

## 예시 2: 항해 화면 개선

```text
game.tscn에 더 명확한 보트 앞부분과 바다 느낌을 추가해줘.
실사풍이 아니라 파스텔톤 Godot 기본 UI/2D 표현으로 충분해.
전투, 실패, 결제, 광고는 추가하지 마.
```

## 예시 3: 앨범 구조 개선

```text
album.tscn을 사진, 풍경, 편지 탭 구조로 정리해줘.
초기에는 실제 이미지 저장 없이 텍스트 기록만 보여줘.
Godot에서 실행 가능한 상태를 유지해줘.
```
